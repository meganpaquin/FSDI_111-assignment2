All About SQL

Available options:

  • mySQL
  • Oracle
  • Microsoft SQL
  • PstgreSQL
  • Amazon Relational Database Service

SQL is a language that is consistent across the above platforms. This standardizes how to add, read, change, and delete data. You only have to learn the process once to use all SQL databases

  1. Create a new table
    2.                 
                    sqlite> CREATE TABLE user(
                        ...> id INTEGER PRIMARY KEY AUTOINCREMENT,
                        ...> first_name VARCHAR(45),
                        ...> last_name VARCHAR(45),
                        ...> hobbies TEXT);
  2. Add data to a table
    3.                 
                    sqlite> INSERT INTO user(
                        ...> first_name,
                        ...> last_name,
                        ...> hobbies
                        ...> ) VALUES (
                        ...> "Megan",
                        ...> "Paquin",
                        ---> "Crafts");
  3. Update data in a table
    4.                 
                    sqlite> UPDATE user SET hobbies="Hiking" WHERE id=1;
  4. Read data from a table
    5.                 
                    sqlite> SELECT * FROM user WHERE last_name="Paquin";
  5. Delete data from a table
    6.                 
                    sqlite> DELETE FROM user WHERE id=3;
